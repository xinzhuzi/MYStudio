# 2D经典圆线漫画 通用风格说明

经典圆线科幻漫画风格，强调圆润结构、大眼和复古未来感。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：classic round-line retro manga
- **视觉要点**：large expressive eyes, rounded face, simple futuristic prop, iconic silhouette
- **光影方案**：clean flat light, vintage print tone, bright clarity
- **质感锚点**：smooth black line, simple shading, old comic texture
- **色彩基线**：black, white, sky blue

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- modern hyper-detailed anime, sharp realistic rendering, 3D。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“classic round-line retro manga”的整体气质，通过“large expressive eyes, rounded face, simple futuristic prop, iconic silhouette”建立识别度，再用“clean flat light, vintage print tone, bright clarity”控制情绪和镜头完成度。
