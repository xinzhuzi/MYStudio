# 2D自然手绘动画 通用风格说明

自然系手绘动画通用风格，强调温柔人物、乡野背景和水彩般空气感。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：hand-painted nature 2D animation
- **视觉要点**：gentle character, lush countryside, peaceful daily life, charming background
- **光影方案**：soft daylight, diffused cloud light, calm atmosphere
- **质感锚点**：watercolor-like background, warm hand-drawn line, organic texture
- **色彩基线**：moss green, cream white, soft sky blue

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- sharp digital neon, 3D CGI, horror darkness。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“hand-painted nature 2D animation”的整体气质，通过“gentle character, lush countryside, peaceful daily life, charming background”建立识别度，再用“soft daylight, diffused cloud light, calm atmosphere”控制情绪和镜头完成度。
