# 导演规划 · 2D简单线条

## 一、色调体系

- **色彩基线**：black, white, single accent color。
- **情绪绑定**：温柔段落降低对比，冲突段落强化明暗，高潮段落提高主体边缘光。
- **禁用色域**：messy sketch, dense background, 3D shading, realism。

## 二、光影方案

- **主光方案**：plain even light, no heavy shadow, graphic clarity。
- **层次原则**：主体、背景和道具要有明确层次，不允许画面糊成一片。
- **媒介边界**：全片保持 2D 动画 / 插画，不混入错媒介词。

## 三、质感方向

- **质感锚点**：thin black line, vector-like edge, minimal fill。
- **提示词基线**：(best quality, masterpiece, high detailed:1.2), (minimal clean line art:1.3), (continuous line drawing, elegant figure, simple object, blank composition:1.18), (plain even light, no heavy shadow, graphic clarity:1.1), thin black line, vector-like edge, minimal fill, clean composition, readable silhouette, high detail, finished illustration。
- **反向规避**：(worst quality, low quality:1.4), blurry, messy lineart, bad anatomy, 3D render, photorealistic, CGI, watermark, signature, text, messy sketch, dense background, 3D shading, realism。

## 四、段落规划

| 段落类型 | 视觉策略 |
|---|---|
| 开场 | 用代表性场景建立 2D简单线条 的风格基线 |
| 人物登场 | 强化主体轮廓、服饰/材质和面部识别点 |
| 冲突段落 | 增强构图张力和光影对比，但不破坏身份连续 |
| 收束段落 | 减少复杂元素，回到清晰、稳定、可延续的视觉基线 |
