# 2D简笔画 通用风格说明

极简简笔画风格，强调白底、手绘线条和低复杂度表达。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：minimalist stick figure doodle
- **视觉要点**：simple stick figure, sketchbook charm, clean blank space, cute expression
- **光影方案**：flat paper light, no complex shadow, minimal contrast
- **质感锚点**：hand-drawn pencil line, plain white background, sparse detail
- **色彩基线**：black line, white, tiny color accent

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- complex realistic detail, filled color, 3D shading。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“minimalist stick figure doodle”的整体气质，通过“simple stick figure, sketchbook charm, clean blank space, cute expression”建立识别度，再用“flat paper light, no complex shadow, minimal contrast”控制情绪和镜头完成度。
