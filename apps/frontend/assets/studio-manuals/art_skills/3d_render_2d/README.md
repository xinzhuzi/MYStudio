# 3D渲染2D 通用风格说明

三渲二通用风格，强调3D体积与2D动画线条、赛璐璐阴影的融合。

## 风格边界

- **媒介类型**：3D 渲染
- **核心风格**：anime-inspired cel shaded 3D render
- **视觉要点**：toon linework, cel shaded body, vibrant fantasy setting, clean anime face
- **光影方案**：bright rim light, soft global illumination, colorful sky light
- **质感锚点**：toon material, sharp edge highlights, controlled flat shadow
- **色彩基线**：cyan sky, warm orange, clean white

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“3D 渲染”媒介边界的错媒介输出。
- photorealistic skin, heavy realism, rough sketch。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“anime-inspired cel shaded 3D render”的整体气质，通过“toon linework, cel shaded body, vibrant fantasy setting, clean anime face”建立识别度，再用“bright rim light, soft global illumination, colorful sky light”控制情绪和镜头完成度。
