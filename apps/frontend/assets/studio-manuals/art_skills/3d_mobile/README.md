# 3D手游 通用风格说明

手游级风格化3D，强调清爽材质、可读轮廓和高完成度游戏资产。

## 风格边界

- **媒介类型**：3D 渲染
- **核心风格**：stylized mobile game 3D render
- **视觉要点**：hero character design, clean fantasy outfit, readable game asset silhouette
- **光影方案**：bright outdoor light, soft ambient light, polished game look
- **质感锚点**：optimized clean material, stylized cloth and metal, vivid but controlled color
- **色彩基线**：blue sky, fresh green, heroic gold

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“3D 渲染”媒介边界的错媒介输出。
- photorealistic noise, rough sketch, pixelated low quality。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“stylized mobile game 3D render”的整体气质，通过“hero character design, clean fantasy outfit, readable game asset silhouette”建立识别度，再用“bright outdoor light, soft ambient light, polished game look”控制情绪和镜头完成度。
