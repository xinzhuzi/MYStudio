# 2D电影 通用风格说明

动画电影级通用风格，强调细致背景、情绪光影和大银幕构图。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：high budget 2D animated movie still
- **视觉要点**：detailed background, emotional sky, cinematic character framing, atmospheric depth
- **光影方案**：dramatic sunset light, layered clouds, soft glow
- **质感锚点**：painterly background, clean character line, film-like composition
- **色彩基线**：golden sunset, deep blue, soft cloud white

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- simple flat cartoon, low resolution, dull color。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“high budget 2D animated movie still”的整体气质，通过“detailed background, emotional sky, cinematic character framing, atmospheric depth”建立识别度，再用“dramatic sunset light, layered clouds, soft glow”控制情绪和镜头完成度。
