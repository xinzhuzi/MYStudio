# 日式3D渲染2D 通用风格说明

日式三渲二通用风格，强调动态姿态、锐利赛璐璐阴影和强镜头张力。

## 风格边界

- **媒介类型**：3D 渲染
- **核心风格**：Japanese cel shaded 3D action render
- **视觉要点**：sharp anime silhouette, dynamic camera angle, bold costume shapes, action pose
- **光影方案**：hard rim light, high contrast stage lighting, motion streaks
- **质感锚点**：crisp toon shader, clear line accents, stylized material breakups
- **色彩基线**：electric blue, black, orange accent

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“3D 渲染”媒介边界的错媒介输出。
- photorealistic rendering, dull flat color, western cartoon softness。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“Japanese cel shaded 3D action render”的整体气质，通过“sharp anime silhouette, dynamic camera angle, bold costume shapes, action pose”建立识别度，再用“hard rim light, high contrast stage lighting, motion streaks”控制情绪和镜头完成度。
