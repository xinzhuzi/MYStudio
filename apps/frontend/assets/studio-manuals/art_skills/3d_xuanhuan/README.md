# 3D玄幻 通用风格说明

东方玄幻题材的通用3D风格，强调仙气、层次、体积光和华丽服饰。

## 风格边界

- **媒介类型**：3D 渲染
- **核心风格**：Chinese fantasy 3D animation render
- **视觉要点**：traditional oriental robes, embroidered fabric, layered mountains, spiritual atmosphere
- **光影方案**：soft volumetric fog, cinematic backlight, glowing aura
- **质感锚点**：polished PBR cloth, fine metal ornaments, ethereal depth
- **色彩基线**：misty jade, cloud white, warm gold

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“3D 渲染”媒介边界的错媒介输出。
- western fantasy, modern city, neon sci-fi。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“Chinese fantasy 3D animation render”的整体气质，通过“traditional oriental robes, embroidered fabric, layered mountains, spiritual atmosphere”建立识别度，再用“soft volumetric fog, cinematic backlight, glowing aura”控制情绪和镜头完成度。
