# 水墨国风约束 · 水墨国风修仙 · 技法参考

---

## 一、色调体系与画面基调

- **色调基底** — 全片以核心色盘为基底：宣纸白（C1 #F5F0E8）、青绿（C2 #5B8C7A）、朱红（C3 #B8453A）、靛蓝（C4 #2B4570）、旧金（C5 #A8883C）、浓墨（C6 #1A1A1A）、淡墨（C7 #6B6B6B）、赭石（C8 #8B5E3C）、暗金（C9 #7A6528）、血红（C10 #8B1A1A），整体色温偏冷中性（4800-5200K），饱和度低（25-45%），呈现水墨淡彩低饱和的修仙古韵画面
- **冷暖叙事对比** — 暖色（朱红 C3、旧金 C5、赭石 C8、暗金 C9）作为正向情感/庄严/突破的视觉信号；冷色（靛蓝 C4、浓墨 C6、青绿 C2）作为冲突/肃杀/幽冥的视觉信号。冷暖切换应与故事弧线同步，而非随意混用
- **冷暖分配策略** — 斗法肃杀/心魔侵蚀段落使用冷色调（靛蓝 + 浓墨 + 青绿），禁用朱红/旧金暖光；修炼悟道/日常修行段落使用柔和暖调（宣纸白 + 青绿 + 赭石）；宗门大典/渡劫突破段落使用朱红 + 旧金 + 暗金高光
- **色盘先行原则** — 段落规划需先绑定情绪场景（修炼悟道/宗门盛典/斗法对决/日常修行/冥界幽境/渡劫突破等），再确定主色+辅色与光影方案，避免"剧情对了但情绪不对色"
- **禁用色域** — 高饱和动漫色、荧光色、霓虹色、赛璐璐色块均与本风格不兼容。色相偏移容差 ±6°，饱和度偏移 ±8%，明度偏移 ±10%

## 二、光影方案体系

- **光影即叙事** — 7 套光影方案对应不同情绪段落，导演规划阶段应在段落层面确定光影基调方向，而非逐镜指定
- **水墨光影为锚** — 所有光影方案必须保持水墨晕染质感，阴影使用淡墨层叠渲染，禁止纯黑硬阴影，允许灵光/景深虚化等水墨渲染技法
- **光源角度** — 正给镜头默认 38.5° 斜射散光（水墨晕染质感），侧给镜头可适当调整

| 光影方案 | 方案名 | 色调倾向 | 适用情绪 |
|---|---|---|---|
| A | 散光清透 | 宣纸白底 + 青绿微暖 | 日常修行、清修悟道、宁静淡泊 |
| B | 仙气浩荡 | 宣纸白 + 青绿 + 旧金边缘光 | 修炼突破、灵气爆发、仙道意境 |
| C | 灵火暖影 | 赭石暖主导 + 浓墨暗部 | 丹房炼丹、洞府夜修、灵火通明 |
| D | 月华冷辉 | 靛蓝 + 宣纸白冷光 | 月下悟道、孤修独处、清冷意境 |
| E | 窗纱灵光 | 宣纸白底 + 侧光斑驳 | 藏经阁/静室、日间修行、静谧 |
| F | 大典辉煌 | 朱红 + 旧金高光 + 浓墨暗部 | 宗门大典、飞升盛典、庄严肃穆 |
| G | 冷调肃杀 | 靛蓝 + 浓墨 + 38.5°侧射冷光，无暖色 | 斗法对决、心魔侵蚀、杀劫降临 |

- **冷暖光分配** — 暖色光（赭石暖/朱红/旧金）适用于正向情感/庄严段落；斗法肃杀/心魔段落一律冷光。导演可根据叙事需要调整冷暖切换点
- **氛围方向映射** — 每场戏的氛围方向应能映射到上述光影方案（A-G）的某一方向，确保视觉一致性

## 三、质感方向

- **水墨晕染为锚** — 水墨国风的核心：工笔线描、写意泼墨、宣纸肌理、墨色浓淡干湿、禁止赛璐璐色块/色块断层
- **工笔写意融合 + 电影质感** — 这是画面质感的锚点。不是写实摄影，不是 3D CG，不是日式动画，是传统水墨画技法加修仙美学的融合
- **材质水墨化** — 所有服化道的材质必须以水墨技法呈现：灵丝的光泽用淡墨渲染、灵金的反光用飞白笔触、灵玉的润感用水墨晕染。禁止"赛璐璐感"和"3D 感"
- **皮肤质感** — 玉白基调、工笔淡染、内蕴微光、温润质感、宣纸肌理。禁止哑光/死白/蜡感/油光/过曝/赛璐璐平涂
- **水墨不等于扁平** — 水墨国风修仙强调在工笔线描基础上通过墨色浓淡、留白、景深虚化、灵光等手段增加空间感与电影感，而非扁平无纵深的简笔画风格

## 四、水墨国风场景空间元素

水墨修仙世界观特有的场景元素及其视觉叙事功能：

- **灵纱/屏风/阵纹门框/雕花窗棂** — 天然的框架式构图道具，制造"看不透"的层次感与空间纵深，以工笔线描呈现精致灵纹细节
- **灵山/灵湖/灵雾/青绿山水** — 留白构图的天然载体，景即情：灵山巍峨 = 道心坚定、灵雾弥漫 = 迷茫困惑、落花飘零 = 道侣离别、祥云缭绕 = 突破在即
- **灵火/月华/窗光/灵灯** — 修仙世界的光源载体，灵火 = 暖/炼丹（方案C）、月华 = 冷/悟道（方案D）、窗光 = 日常/静修（方案E）、灵灯 = 大典/宗门（方案F）
- **飞檐斗拱/灵石宫墙/阵纹石桥/灵木家具** — 建筑元素以水墨工笔呈现，细节丰富、工笔线描清晰，材质带自然磨损/岁月包浆感，禁止全新无瑕的"塑料感"
- **段落间用场景空镜过渡** — 本风格有丰富的场景资产（不同时段/天候/景别变体），段落衔接建议用场景空镜做情绪缓冲，不要硬切
- **转折点用视觉而非台词** — 优先使用画面手段（光影突变、景别跳切、空镜隐喻）而非依赖对白解释
- **前中后景层次** — 所有场景必须具备前/中/后景，杜绝扁平，室外必须包含水墨空气透视（远山如黛）

## 五、修仙乐器与环境音

修仙世界观下的声音元素约束：

### 乐器选择

- **洞箫** — 悟道、孤修、清冷段落的核心乐器，与月下悟道/仙道意境氛围高度适配
- **古琴** — 开场定调 / 平稳段落 / 修炼意境，与洞箫搭配使用
- **埙** — 苍凉、悲壮、渡劫段落，土之声与天地共鸣
- **编钟** — 宗门大典 / 飞升盛典 / 庄严肃穆段落
- **琵琶** — 紧张、急促段落的点缀，斗法场景的节奏推动器，不作主导
- **大鼓/战鼓** — 斗法高潮、渡劫雷声、天地异象的节奏骨架
- 弦乐铺底可增加电影感但不宜喧宾夺主

### 乐器组合策略

| 情绪阶段 | 乐器组合 |
|---|---|
| 平稳/开场/收尾 | 古琴独奏 或 古琴 + 洞箫 |
| 修炼/悟道/意境 | 洞箫 + 古琴 + 轻铃 |
| 悲壮渐浓 | 埙 + 洞箫 |
| 渡劫/命运转折 | 大鼓 + 编钟 + 埙 |
| 紧张/斗法/对决 | 琵琶点缀 + 鼓点 + 弦乐衬底 |
| 日常修行/清修 | 古琴 + 竹笛 |
| 宗门大典/飞升 | 编钟 + 古琴 + 弦乐铺底 |

### 修仙环境音

- **典型环境音层次** — 灵鹤长鸣 / 灵泉潺潺 / 风过灵竹 / 丹炉火响 / 夜雨滴檐 / 衣袂飘动 / 灵铃轻响 / 落花飘零 / 灵气流转微响
- **每场戏标注 1-2 个核心环境音**，帮助后续音效设计。环境音层次越丰富，修仙场景越有沉浸感
