# 分镜提示词 · 3D国风赛博 · 风格专属技法

## 适用范围
本 Skill 专用于**3D国风赛博（新中式赛博朋克）**风格的分镜提示词生成，核心为「中式美学为魂，赛博科幻为形」的有机融合，**同时完美适配两大创作方向**：
1.  传统古风场景赛博化（宫廷、园林、武侠、宗门等古风原生场景）
2.  现代都市场景国风赛博化（商圈、写字楼、公寓、夜市等都市现代场景）
拒绝元素生硬拼接，坚守中式美学内核。
