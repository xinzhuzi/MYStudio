# 2D圆润儿童动画 通用风格说明

圆润儿童向动画风格，强调简单线条、童趣比例和明亮纯色。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：rounded child-friendly 2D animation
- **视觉要点**：simple round character design, friendly face, clean prop shapes, playful room
- **光影方案**：bright even light, cheerful color, soft shadow
- **质感锚点**：clean outline, simple flat color, low detail density
- **色彩基线**：primary blue, warm yellow, clean white

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- complex realistic detail, dark gloomy horror, sharp angular face。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“rounded child-friendly 2D animation”的整体气质，通过“simple round character design, friendly face, clean prop shapes, playful room”建立识别度，再用“bright even light, cheerful color, soft shadow”控制情绪和镜头完成度。
