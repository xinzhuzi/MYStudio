# 全局美学基础 · 2D像素

---
必须严格、完整遵循下方全部风格约束与全局规则，并严格按提示词模板格式生成提示词；仅输出提示词正文，不得附加任何解释、说明、注释、标题或其他额外文本。

## 一、风格基因

| 维度 | 定义 |
|---|---|
| 一级风格 | 2D像素 |
| 媒介类型 | 2D 动画 / 插画 |
| 核心锚点 | clean pixel art style |
| 视觉要点 | 16-bit sprite look, tile-based environment, readable silhouette, retro game asset |
| 情绪基调 | 像素艺术风格，强调格点清晰、有限色板和复古游戏资产感。 |
| 质感锚词 | hard pixel edges, dithering, limited palette |

## 二、全局色彩与光影

| 项目 | 约束 |
|---|---|
| 色彩基线 | retro green, blue, magenta |
| 主光方案 | simple light direction, crisp pixel shadow, no blur |
| 画面层次 | 主体清晰、前中后景可读、焦点明确 |
| 质量锚定 | (best quality, masterpiece, high detailed:1.2), (clean pixel art style:1.3), (16-bit sprite look, tile-based environment, readable silhouette, retro game asset:1.18), (simple light direction, crisp pixel shadow, no blur:1.1), hard pixel edges, dithering, limited palette, clean composition, readable silhouette, high detail, finished illustration |
| 反向规避 | (worst quality, low quality:1.4), blurry, messy lineart, bad anatomy, 3D render, photorealistic, CGI, watermark, signature, text, anti-aliased smooth line, vector art, 3D realism, blur |

## 三、必守规则

| 编号 | 规则 |
|---|---|
| R1 | 必须声明“2D像素”和“clean pixel art style”作为风格锚点 |
| R2 | 必须保持“2D 动画 / 插画”媒介边界，不混入错媒介词 |
| R3 | 必须包含“16-bit sprite look, tile-based environment, readable silhouette, retro game asset”中至少 2 个视觉识别点 |
| R4 | 必须说明光源或光影方向：simple light direction, crisp pixel shadow, no blur |
| R5 | 必须把质量锚点与主体内容融合，不输出解释性说明 |

## 四、严禁项

| 编号 | 严禁内容 |
|---|---|
| X1 | 直接写入具体作品、作者、厂牌或版权角色名称 |
| X2 | anti-aliased smooth line, vector art, 3D realism, blur |
| X3 | 低质量、模糊、变形、裁切、文字、水印、签名 |
| X4 | 人物身份漂移、服装不一致、手部结构错误、场景空间混乱 |
