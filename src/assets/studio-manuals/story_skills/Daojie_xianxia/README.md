# 道劫 · 水墨修仙 · 导演叙事手法技能包

## 简介

本技能包为**《道劫》**专属导演叙事手法参考，基于水墨国风视觉体系，涵盖从宏观叙事规划到微观分镜执行的全流程指导。本技能包深度融合道劫独有的三族灵气体系、境界突破叙事、剑修战斗美学与万劫圣宗世界观，为导演提供精准的叙事工具。

## 核心理念

- **水墨即天道** — 所有视觉表达以水墨画为根基：泼墨写意传递磅礴、工笔细描刻画灵韵、留白即虚空、晕染即灵气。宣纸质感贯穿始终
- **三色灵气即种族** — 人族朱红、妖族青绿、鬼族靛蓝，三色墨韵是道劫世界的视觉基因。灵气不是特效，是水墨晕染的自然延伸
- **万道归真** — 叙事的终极方向是"万族归一、万道归真"。从种族隔离到众生同火，从个人修行到天道重塑
- **劫即叙事** — "劫"是道劫的核心叙事引擎：心劫（情感）→ 身劫（肉体）→ 道劫（信念）→ 天劫（天罚）→ 万劫（众生）。每一劫都是主角与天道的对话
- **孤剑问道** — 苏无忌的底色是"一人一剑问天道"。从道口镇少年到万道归真，孤独不是悲剧，是求道者的必然
- **留白即大道** — 最深的道、最浓的情，都不靠说——靠一柄归元剑、一壶浊酒、一场血雨中的沉默

## 使用方式

本技能包作为 ToonFlow 导演 AI 的叙事手法参考，在道劫故事创作流程中自动加载，指导：

1. **叙事规划** — 确定道劫内核、灵气视觉体系、境界突破节奏、剑修战斗美学与音乐方向
2. **分镜表生成** — 根据叙事规划输出具体的水墨景别、灵气运镜、突破时长与转场方案

两个文件分工明确、逐层递进，从"为什么这样拍"到"具体怎么拍"形成完整闭环。所有视觉描述均以水墨国风为基底，拒绝赛璐璐动漫风。
