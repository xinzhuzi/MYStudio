# 3D方块世界 通用风格说明

体素方块世界风格，强调方块结构、像素化体积和玩具感空间。

## 风格边界

- **媒介类型**：3D 渲染
- **核心风格**：voxel block world 3D art
- **视觉要点**：cubic character, blocky trees, grid-based village, isometric readability
- **光影方案**：bright daylight, crisp shadow, cheerful atmosphere
- **质感锚点**：voxel cubes, pixel-like material, clean toy blocks
- **色彩基线**：grass green, clear blue, flower red

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“3D 渲染”媒介边界的错媒介输出。
- round organic forms, smooth realistic texture, blur。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“voxel block world 3D art”的整体气质，通过“cubic character, blocky trees, grid-based village, isometric readability”建立识别度，再用“bright daylight, crisp shadow, cheerful atmosphere”控制情绪和镜头完成度。
