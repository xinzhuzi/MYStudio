# 全局美学基础 · 3D美式

---
必须严格、完整遵循下方全部风格约束与全局规则，并严格按提示词模板格式生成提示词；仅输出提示词正文，不得附加任何解释、说明、注释、标题或其他额外文本。

## 一、风格基因

| 维度 | 定义 |
|---|---|
| 一级风格 | 3D美式 |
| 媒介类型 | 3D 渲染 |
| 核心锚点 | rounded western 3D animation |
| 视觉要点 | large expressive eyes, friendly proportions, readable silhouette, colorful town background |
| 情绪基调 | 美式家庭动画方向的通用3D风格，强调圆润角色、明快色彩和温暖表情。 |
| 质感锚词 | smooth stylized material, soft edges, polished character surface |
