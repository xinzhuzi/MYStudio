# 3D美式 通用风格说明

美式家庭动画方向的通用3D风格，强调圆润角色、明快色彩和温暖表情。

## 风格边界

- **媒介类型**：3D 渲染
- **核心风格**：rounded western 3D animation
- **视觉要点**：large expressive eyes, friendly proportions, readable silhouette, colorful town background
- **光影方案**：warm key light, soft fill light, cheerful daylight
- **质感锚点**：smooth stylized material, soft edges, polished character surface
- **色彩基线**：sunny amber, sky blue, soft coral

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“3D 渲染”媒介边界的错媒介输出。
- dark gritty realism, horror mood, hard realistic skin。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“rounded western 3D animation”的整体气质，通过“large expressive eyes, friendly proportions, readable silhouette, colorful town background”建立识别度，再用“warm key light, soft fill light, cheerful daylight”控制情绪和镜头完成度。
