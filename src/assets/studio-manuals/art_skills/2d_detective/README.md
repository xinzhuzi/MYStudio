# 2D日式侦探 通用风格说明

日式侦探动画通用风格，强调悬疑氛围、清晰人物和90年代动画质感。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：Japanese detective anime style
- **视觉要点**：sharp facial features, mystery mood, school and city setting, clue-focused framing
- **光影方案**：cool evening light, dramatic interior shadow, suspense tone
- **质感锚点**：clean cel line, restrained color, classic animation finish
- **色彩基线**：navy, beige, warm lamp light

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- fantasy magic excess, modern glossy 3D, cute chibi。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“Japanese detective anime style”的整体气质，通过“sharp facial features, mystery mood, school and city setting, clue-focused framing”建立识别度，再用“cool evening light, dramatic interior shadow, suspense tone”控制情绪和镜头完成度。
