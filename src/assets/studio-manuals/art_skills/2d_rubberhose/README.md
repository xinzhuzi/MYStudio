# 2D橡皮管动画 通用风格说明

复古橡皮管动画风格，强调摆动肢体、派眼和黑白胶片质感。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：1930s rubber hose cartoon
- **视觉要点**：bouncy limbs, pie-cut eyes, vintage mascot character, simple stage
- **光影方案**：old film light, vignette, soft grain
- **质感锚点**：inked black line, limited palette, analog film texture
- **色彩基线**：black, cream, warm grey

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- modern anime, realistic 3D, stiff motion。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“1930s rubber hose cartoon”的整体气质，通过“bouncy limbs, pie-cut eyes, vintage mascot character, simple stage”建立识别度，再用“old film light, vignette, soft grain”控制情绪和镜头完成度。
