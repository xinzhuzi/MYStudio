# 真人复古武侠 通用风格说明

复古武侠真人电影风格，强调实景打斗、布景年代感和江湖气。

## 风格边界

- **媒介类型**：真人摄影
- **核心风格**：vintage live-action wuxia cinema
- **视觉要点**：martial arts stance, old inn, forest duel, practical costume
- **光影方案**：hard side light, dusty backlight, retro film contrast
- **质感锚点**：film grain, worn fabric, real weapon surface
- **色彩基线**：earth brown, faded red, cool night blue

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“真人摄影”媒介边界的错媒介输出。
- modern clothing, sci-fi tech, CGI magic excess, anime。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“vintage live-action wuxia cinema”的整体气质，通过“martial arts stance, old inn, forest duel, practical costume”建立识别度，再用“hard side light, dusty backlight, retro film contrast”控制情绪和镜头完成度。
