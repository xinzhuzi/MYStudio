# 2D水彩 通用风格说明

水彩插画风格，强调湿画法、柔边、纸纹和梦幻色彩。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：watercolor illustration style
- **视觉要点**：soft edge landscape, translucent color wash, dreamy character, paper texture
- **光影方案**：diffused daylight, gentle bloom, low contrast
- **质感锚点**：wet-on-wet pigment, visible paper grain, feathered edge
- **色彩基线**：sky blue, rose, pale yellow

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- hard digital flat color, vector edge, 3D realism。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“watercolor illustration style”的整体气质，通过“soft edge landscape, translucent color wash, dreamy character, paper texture”建立识别度，再用“diffused daylight, gentle bloom, low contrast”控制情绪和镜头完成度。
