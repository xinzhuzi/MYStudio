# 2D像素 通用风格说明

像素艺术风格，强调格点清晰、有限色板和复古游戏资产感。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：clean pixel art style
- **视觉要点**：16-bit sprite look, tile-based environment, readable silhouette, retro game asset
- **光影方案**：simple light direction, crisp pixel shadow, no blur
- **质感锚点**：hard pixel edges, dithering, limited palette
- **色彩基线**：retro green, blue, magenta

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- anti-aliased smooth line, vector art, 3D realism, blur。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“clean pixel art style”的整体气质，通过“16-bit sprite look, tile-based environment, readable silhouette, retro game asset”建立识别度，再用“simple light direction, crisp pixel shadow, no blur”控制情绪和镜头完成度。
