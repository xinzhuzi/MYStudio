# 法宝衍生状态生成 · 约束手册

---

## 一、衍生原则

1. **造型锚定** — 法宝核心造型/轮廓在所有状态中可识别
2. **状态可读** — 状态差异必须一目了然，观众能立即区分
3. **叙事服务** — 每种状态变体服务于特定剧情节点
4. **渐进退化** — 损伤/老化状态应有合理的物理逻辑
5. **纯法宝独立展示** — 画面中只能出现法宝本身，严禁出现任何人物、手部、肢体，法宝不可处于被持有/佩戴/握持状态，必须以静物陈列方式独立呈现

## 三、状态变体画面规范

### 单状态图

| 项目 | 约束 |
|---|---|
| 背景 | 宣纸白底色 #F5F0E8（与设定图一致） |
| 光线 | 均匀照明，无硬阴影 |
| 角度 | 与原设定图正面图一致 |
| 比例 | 法宝占画面主体 70%+ |

### 状态对比图

| 项目 | 约束 |
|---|---|
| 布局 | 同一画面并排展示 2-3 种状态 |
| 标注 | 每种状态下方标注状态名 |
| 一致性 | 角度/光线/背景完全一致，仅状态不同 |

## 六、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 法宝核心造型/轮廓在所有状态中可识别 |
| R2 | 状态变化须符合物理逻辑与修仙世界观 |
| R3 | 必须使用四宫格（2×2）布局：正面 + 侧面 + 背面 + 特写 |
| R4 | 必须指定「宣纸白底色背景」，均匀散光，无硬阴影 |
| R5 | 必须包含「水墨国风 + 工笔线描」关键词 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 状态变化后法宝不可识别 |
| X2 | 违反物理逻辑的损伤（灵玉生锈等） |
| X3 | 过度血腥/恐怖的损伤描绘 |
| X4 | 出现任何人物形象，包括全身、半身、局部（手、手指、手臂等肢体） |
| X5 | 法宝处于被持有、握持、佩戴、使用中的状态 |
| X6 | 出现暗示人物存在的元素（如手持痕迹、佩戴视角、使用姿态） |
