# 道劫 · 水墨国风修仙风格说明

本风格专为《道劫》修仙题材打造，所有美术提示词、规范和生成内容均严格限定于：

- **水墨国风动画**：以中国传统水墨画为基底的修仙动画风格，涵盖修仙、斗法、渡劫等题材
- **工笔写意融合**：工笔线描勾勒形体结构，写意泼墨渲染气韵意境，二者有机结合
- **传统水墨技法**：皴擦点染、浓淡干湿、留白飞白等传统国画技法的数字化表达
- **修仙古韵**：仙道意境、天人合一、道法自然的东方玄学美学表达
- **宣纸绢本质感**：画面呈现宣纸/绢本的纤维肌理、墨色渗透、水痕晕散效果
- **水墨电影质感**：电影级构图、水墨光影层次、动态张力、灵气氛围

## 适用范围

- 修仙题材的AI美术生成（角色、场景、法宝、分镜等）
- 需要水墨国风、工笔写意、修仙意境的视觉创作
- 修仙动画、仙侠游戏美术、概念设计、水墨插画创作

## 严禁内容

- 完全写实摄影/3D写实渲染
- 日式赛璐璐平涂/动漫渲染
- 赛博朋克/科幻/过度现代元素
- 粗劣线条/模糊画质/低精度建模
- 色彩混乱/风格不统一/审美崩坏

## 风格体验

在本风格下，您将体验到：

- 角色造型以工笔白描勾勒，线条如铁线银钩，气韵生动
- 场景充满水墨意境，山水云雾以写意泼墨呈现，留白深远
- 法宝器物以工笔重彩描绘，材质质感古朴，灵光内蕴
- 分镜与渲染均强调水墨电影感、修仙古韵、传统国画技法
