# 毛绒定格动画 通用风格说明

毛绒定格风格，强调羊毛毡纤维、柔软触感和手工温度。

## 风格边界

- **媒介类型**：定格动画
- **核心风格**：needle-felt stop-motion animation
- **视觉要点**：wool character, fuzzy miniature prop, handmade craft environment
- **光影方案**：warm diffuse light, soft fabric shadow, cozy atmosphere
- **质感锚点**：visible wool fibers, felt texture, stitched handmade detail
- **色彩基线**：warm beige, wool brown, pastel accent

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“定格动画”媒介边界的错媒介输出。
- hard plastic, shiny metal, 2D anime, photorealistic human。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“needle-felt stop-motion animation”的整体气质，通过“wool character, fuzzy miniature prop, handmade craft environment”建立识别度，再用“warm diffuse light, soft fabric shadow, cozy atmosphere”控制情绪和镜头完成度。
