# 导演规划 · 毛绒定格动画

## 一、色调体系

- **色彩基线**：warm beige, wool brown, pastel accent。
- **情绪绑定**：温柔段落降低对比，冲突段落强化明暗，高潮段落提高主体边缘光。
- **禁用色域**：hard plastic, shiny metal, 2D anime, photorealistic human。

## 二、光影方案

- **主光方案**：warm diffuse light, soft fabric shadow, cozy atmosphere。
- **层次原则**：主体、背景和道具要有明确层次，不允许画面糊成一片。
- **媒介边界**：全片保持 定格动画，不混入错媒介词。

## 三、质感方向

- **质感锚点**：visible wool fibers, felt texture, stitched handmade detail。
- **提示词基线**：(best quality, masterpiece, high detailed:1.2), (needle-felt stop-motion animation:1.3), (wool character, fuzzy miniature prop, handmade craft environment:1.18), (warm diffuse light, soft fabric shadow, cozy atmosphere:1.1), visible wool fibers, felt texture, stitched handmade detail, macro photography, tactile detail, frame-by-frame charm, high detail。
- **反向规避**：(worst quality, low quality:1.4), fluid CGI animation, 2D anime, photorealistic human scale, smooth digital texture, bad anatomy, watermark, signature, text, hard plastic, shiny metal, 2D anime, photorealistic human。

## 四、段落规划

| 段落类型 | 视觉策略 |
|---|---|
| 开场 | 用代表性场景建立 毛绒定格动画 的风格基线 |
| 人物登场 | 强化主体轮廓、服饰/材质和面部识别点 |
| 冲突段落 | 增强构图张力和光影对比，但不破坏身份连续 |
| 收束段落 | 减少复杂元素，回到清晰、稳定、可延续的视觉基线 |
