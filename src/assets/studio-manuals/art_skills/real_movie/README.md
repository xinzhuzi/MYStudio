# 真人电影 通用风格说明

真人电影剧照风格，强调真实摄影、电影调色和镜头景深。

## 风格边界

- **媒介类型**：真人摄影
- **核心风格**：live-action cinematic movie still
- **视觉要点**：real actor presence, natural costume, film set atmosphere, grounded composition
- **光影方案**：35mm film light, dramatic key light, color graded frame
- **质感锚点**：film grain, real skin pores, optical lens texture
- **色彩基线**：teal orange, neutral skin, dark shadow

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“真人摄影”媒介边界的错媒介输出。
- 3D render, CGI, anime, illustration, artificial plastic face。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“live-action cinematic movie still”的整体气质，通过“real actor presence, natural costume, film set atmosphere, grounded composition”建立识别度，再用“35mm film light, dramatic key light, color graded frame”控制情绪和镜头完成度。
