# 2D简单线条 通用风格说明

极简线稿风格，强调连续线、白底和优雅留白。

## 风格边界

- **媒介类型**：2D 动画 / 插画
- **核心风格**：minimal clean line art
- **视觉要点**：continuous line drawing, elegant figure, simple object, blank composition
- **光影方案**：plain even light, no heavy shadow, graphic clarity
- **质感锚点**：thin black line, vector-like edge, minimal fill
- **色彩基线**：black, white, single accent color

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“2D 动画 / 插画”媒介边界的错媒介输出。
- messy sketch, dense background, 3D shading, realism。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“minimal clean line art”的整体气质，通过“continuous line drawing, elegant figure, simple object, blank composition”建立识别度，再用“plain even light, no heavy shadow, graphic clarity”控制情绪和镜头完成度。
