# 分镜表设计 · 2D简单线条

## 一、分镜表定位

分镜表负责把剧本转为镜头语言，本文件限定 2D简单线条 在镜头、光影、材质和连续性上的写法。

## 二、镜头字段规则

| 字段 | 写法 |
|---|---|
| 画面描述 | 必须含主体、动作、空间、光影和 2D简单线条 风格锚点 |
| 风格提示词 | minimal clean line art; continuous line drawing, elegant figure, simple object, blank composition; thin black line, vector-like edge, minimal fill |
| 光影 | plain even light, no heavy shadow, graphic clarity |
| 负向规避 | (worst quality, low quality:1.4), blurry, messy lineart, bad anatomy, 3D render, photorealistic, CGI, watermark, signature, text, messy sketch, dense background, 3D shading, realism |
| 连续性 | 人物身份、服装、场景结构、道具位置必须跨镜头一致 |

## 三、运镜建议

- 角色镜头优先保持主体清楚，避免镜头运动遮挡身份识别点。
- 场景镜头优先体现前中后景，给后续视频生成留出空间。
- 动作镜头要写清起点、过程、落点，避免模型自行补出不相关动作。

## 四、硬禁忌

- 不写具体作品、作者、厂牌或版权角色名称。
- 不把 2D简单线条 混成其他媒介。
- 不输出水印、字幕、标题叠字、UI 元素。
