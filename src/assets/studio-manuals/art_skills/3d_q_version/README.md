# 3DQ版 通用风格说明

潮玩感Q版三维风格，强调可爱比例、软光、干净材质和收藏级展示。

## 风格边界

- **媒介类型**：3D 渲染
- **核心风格**：chibi collectible 3D toy render
- **视觉要点**：super deformed body, oversized head, cute face, miniature scene
- **光影方案**：soft studio lighting, gentle rim light, clean shadow
- **质感锚点**：smooth toy material, rounded surface, tactile miniature detail
- **色彩基线**：cream beige, pastel green, warm peach

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“3D 渲染”媒介边界的错媒介输出。
- realistic adult proportion, rough material, scary mood。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“chibi collectible 3D toy render”的整体气质，通过“super deformed body, oversized head, cute face, miniature scene”建立识别度，再用“soft studio lighting, gentle rim light, clean shadow”控制情绪和镜头完成度。
