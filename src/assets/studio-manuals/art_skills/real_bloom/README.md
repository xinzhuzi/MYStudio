# 真实光晕 通用风格说明

柔光逆光真人摄影风格，强调梦幻高光、真实皮肤和轻柔景深。

## 风格边界

- **媒介类型**：真人摄影
- **核心风格**：dreamy backlit live-action photography
- **视觉要点**：real person, glowing rim light, soft focus portrait, airy environment
- **光影方案**：strong bloom, lens flare, warm backlight, shallow depth of field
- **质感锚点**：film softness, skin texture retained, optical glow
- **色彩基线**：cream, pale gold, soft blue

## 适用范围

- 角色基础形象、角色衍生服化、场景、场景衍生、道具、分镜和视频提示词。
- 可作为项目默认视觉风格，也可复制到“我的风格”后继续细化。
- 适合做统一美术基线，不绑定任何具体作品、作者或厂牌。

## 严禁内容

- 直接照搬具体作品、作者、厂牌或版权角色的名称与固定造型。
- 偏离“真人摄影”媒介边界的错媒介输出。
- harsh contrast, dark gritty mood, anime, 3D render。
- 低质量、模糊、裁切、文字水印、结构错误、身份漂移。

## 风格体验

在本风格下，画面应优先呈现“dreamy backlit live-action photography”的整体气质，通过“real person, glowing rim light, soft focus portrait, airy environment”建立识别度，再用“strong bloom, lens flare, warm backlight, shallow depth of field”控制情绪和镜头完成度。
