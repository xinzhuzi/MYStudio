# 人物基础形象生成 · 2D韩式动画

## 一、基础原则

- 生成 2D 角色设定图，用于角色首次定型。
- 必须保持“premium webtoon style illustration”和“2D 动画 / 插画”媒介边界。
- 人物需具备清晰身份、年龄、性别、五官、体态、发型、基础服装和气质标签。
