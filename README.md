# 漫影工作室

本地优先的 AI 漫剧与短剧制作工作台。

## V1 边界

- 基底来自 Moyin：Electron + React + TypeScript + Zustand + 文件型项目存储。
- 工作流参考 Toonflow：小说导入、Skill 上下文包、剧本草稿、分镜表、制作流、剪辑工作台。
- 本地合成参考 Toonflow/Huobao：通过 Electron main process 调用本机 FFmpeg 生成 track 候选片段，并按选中片段拼接成片。
- 模型暂不执行：配置中心可保存中转站、模型能力和任务绑定，但 V1 只做结构校验和 dry-run。

## 当前入口

进入项目后打开左侧 `工作流`：

1. `小说` 导入 `.txt/.md` 或粘贴正文。
2. `Skill` 生成上下文包并保存人工工作数据。
3. `剧本` 保存剧本草稿。
4. `分镜` 维护 track、时长、素材路径和台词。
5. `剪辑` 用本地 FFmpeg 生成候选片段，选择后拼接成片。
6. `配置` 保存中转站、模型定义和任务绑定。
